# building-cloud-integrations

Written by Michael Meier and Lucas Aebi 


## Usage

Start the server:

```bash
npm run dev
# or
SECRET="secret" npm run server
# or
SECRET="secret" node index.js
```

To run the test
```bash
SECRET="secret" npx mocha
# or 
npm run test
```
